Steps Of Creating Our Project

Analysis Phase

Analysis was done using Power BI, answering 20 questions about the data.

Preprocessing Phase

Due to the presence of the date feature, the data was handled as a time series problem,Data transformation using the discrete cosine transform as this is a time series data we suspected that the data may have been periodic.

Modeling Phase

We tried to use several models to assume that we picked the best model.

Deployment Phase

The selected model has been uploaded to the backend and the front end of the website has been handled after that we have hosted it.
